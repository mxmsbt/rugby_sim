# Upstream

`rugby_sim` started as a fork of `google-research/football`.

- Upstream repository: `https://github.com/google-research/football`
- Fork owner: `mxmsbt`
- Intent: replace football match logic with a rugby union 15v15 engine while
  preserving a practical RL/simulation API and native rendering pipeline

Inherited files remain under their original licenses and notices. New code in
this fork should be marked clearly as RugbySim-specific.
